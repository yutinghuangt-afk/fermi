# 費米模型市場規模分析工具

智慧高軟整合行銷組內部策略工具，用於估算企業數位轉型顧問服務的可獲市場（TAM / SAM / SOM）。

## 用途

- 透過費米估算法，將市場規模拆解為可驗證的假設步驟
- 每個假設值皆可即時調整，右側自動重算結果
- 含競爭者市場扣除邏輯，反映實際可競爭空間

## 使用方式

直接用瀏覽器開啟 `fermi-market-analysis.html`，無需安裝任何套件。

## 檔案說明

| 檔案 | 說明 |
|------|------|
| `fermi-market-analysis.html` | 主工具頁面（費米模型互動計算） |
| `tam-sam-som.html` | TAM/SAM/SOM 靜態摘要報告 |

## 對象

智慧高軟整合行銷組｜內部策略規劃用
